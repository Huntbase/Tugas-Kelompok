Struktur Dasar HTML

Berikut ini ada kode HTMl yang baru saja kita buat:


Kalau kita perhatikan. Struktur dasar kode HTML terdiri dari tiga bagian penting:


  1. Bagian Deklarasi
  2. Bagian Head
  3. Bagian Body
Doctype

kode yang mengindikasikan bahwa dokumen tersebut adalah halaman HTML. Fungsinya adalah memberitahu peramban (browser) bahwa kode yang akan dijalankan adalah kode bahasa markup.


html

berfungsi sebagai pembuka dan penutup dari keseluruhan dokumen bahasa markup. Semua elemen bahasa markup diletakkan di antara tag ini.


head

biasanya berisi elemen-elemen yang tidak terlihat pada halaman web, seperti pemanggilan CSS, pembuatan judul website, favicon, dan lainnya. Semua konten di dalam tag head tidak akan ditampilkan di peramban.

body

Pada bahasa markup berisi elemen-elemen yang akan ditampilkan di halaman web. Ini termasuk paragraf, gambar, dan elemen-elemen lain yang akan terlihat oleh pengguna saat halaman web dibuka di peramban.


Tag HTML

Tag HTML adalah elemen-elemen yang digunakan untuk memberi tahu peramban bagaimana cara merender dan menampilkan konten di halaman web. Beberapa tag bahasa markup umum meliputi:

  • html: Menandai awal dan akhir dari dokumen HTML.
  • head: Berisi informasi meta tentang dokumen, seperti judul, link ke stylesheet, dll.
  • title: Menentukan judul halaman yang akan ditampilkan di bilah judul browser.
  • body: Berisi konten utama dokumen yang akan ditampilkan di halaman web.
  • h1 - h6: Mendefinisikan heading (judul) dengan tingkat prioritas berbeda.
  • p: Menandai paragraf teks.
  • a: Membuat tautan (link) ke halaman atau sumber lain.
  • img: Menyisipkan gambar ke dalam halaman menggunakan URL gambar.
  • ul, ol, li : Membuat daftar tak berurutan (unordered) atau berurutan (ordered) dengan item daftar.
  • div: Mengelompokkan elemen-elemen bahasa markup untuk diberikan gaya atau perilaku tertentu.
  • span: Serupa dengan (div), tetapi untuk kelompok teks atau elemen inline.
  • input: Membuat bidang input seperti teks, checkbox, atau tombol.
  • form: Fungsinya form untuk mengumpulkan input pengguna.
  • button: Berfungsi agar bisa tombol dapat diklik.
  • table, tr, td : Membuat tabel dengan baris dan sel data.
